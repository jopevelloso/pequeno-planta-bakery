# PEQUENO PLANTA - Urban Bakery E-Commerce

This is a responsive, urban-styled e-commerce template for an artisanal bakery.

## Features

-   **Urban Design:** High contrast, bold typography (Oswald/Roboto), and clean layout.
-   **Product Showcase:** Featured products (Bread, Cookies, Cakes) with custom SVG icons.
-   **Responsive:** Adapts to desktop and mobile screens.
-   **No External Image Dependencies:** Uses local SVG assets and Font Awesome CDN.

## Project Structure

-   `index.html`: Main landing page structure.
-   `style.css`: Custom urban styling.
-   `bread.svg`, `cookie.svg`, `cake.svg`: Product icons.
-   `hero.svg`: Hero background image.
-   `logo.png`: Brand logo.

## Usage

1.  Open `index.html` in a web browser.
2.  Customize content in HTML.
3.  Modify styles in `style.css` (variables available at the top).
